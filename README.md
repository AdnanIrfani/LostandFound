# Lost and Found College Portal (Laravel)

## Requirements
- PHP >= 8.1
- Composer
- MySQL
- Node.js (optional)

## Setup Steps

1. Extract project  
2. Open terminal in project folder  
3. Run:
   composer install  
4. Copy env:
   cp .env.example .env  
5. Set database in .env  
6. Generate key:
   php artisan key:generate  
7. Import database  
8. Run:
   php artisan migrate  
9. Start server:
   php artisan serve
